Digital Pen(Pencil): Machine Learning

So, we started by using ESP for gesture recognition. We found some interesting observations while experimenting with it.

Setup







Interesting observations

1) The gesture(writing) need to be fast,else ESP shows warning that sample does not have enough movement.

2) The system produces lot of false positive, the systems accuray can increase if increase the number of sample per category. We tried using 50 sample for each category but the software crashed.

3) Placing the accelerometer on different part of the pencil may result in better performance.

Future Work

1) Create a proper mount for the accelerometer to fit on the pen/pencil, so that we can avoid noise and get some stable reading.

2) Look for othr alternative to ESP which can be used with android.

3) Create better training data to train the model.
