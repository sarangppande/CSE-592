Arduino Box Part 1: Design

The aim of this assignment is to get familiarized with FreeCAD and Inkscape. We designed a box for Arduino Box.

Initial Measurement

We got the Arduino basic measurements from its specification mentioned on the website. For measuring the position of the slot for USB we used FreeCad model of Arduino.









FreeCad

We used the LC interlocking in FreeCad to get the basic shape of the box and to add tabs. We then exported the design as a flattened SVG. Box Design 2-D Export





Inkscape

We used inkspace to add slots for USB and sensors. Final Layout
